# Career Metro

A beautiful, Apple-inspired career mapping visualization for the battery industry.

## Features

- 🎨 Apple Design Principles
- 🗺️ Interactive Metro Map
- 🔗 Skill-based Connections
- 📊 Progressive Career Paths
- 🔍 Search & Filter
- 📱 Responsive Design

## Quick Deploy

### Option 1: Netlify Drop (Fastest - No account needed)

1. Go to [https://app.netlify.com/drop](https://app.netlify.com/drop)
2. Drag and drop the `index.html` file
3. Your site is live instantly!

### Option 2: GitHub Pages

1. Create a new repository on GitHub
2. Upload `index.html`
3. Go to Settings → Pages
4. Select main branch and `/ (root)` folder
5. Your site will be live at `https://yourusername.github.io/repo-name`

### Option 3: Vercel

```bash
npm i -g vercel
vercel
```

## Data Source

Data is dynamically loaded from Google Sheets. Update the `GOOGLE_SHEETS_URL` in `index.html` to change the data source.
